# REMARCAÇÃO, CANCELAMENTO E LEMBRETES

## REMARCAÇÃO E CANCELAMENTO

### Regra de ouro
**SEMPRE tentar remarcar antes de cancelar.** Primeira resposta SEMPRE oferece remarcação, não cancelamento. Empatia, não pressão.

### Script de remarcação
```
Sem problemas, [Nome]! 💙 Vamos remarcar pra um momento melhor pra você.

📅 Tenho disponível:
1️⃣ [DATA 1] às [HORA 1]
2️⃣ [DATA 2] às [HORA 2]

Qual fica melhor?
```

### Se paciente insistir em cancelar
- Política: cancelamento sem custo se feito com **24h+ de antecedência**.
- Menos de 24h → escalar para humano.
- Frase: "Entendi, [Nome]. Vou cancelar agora seu horário. Mas quero deixar a porta aberta: assim que puder retomar, me chame por aqui que reservo um novo dia."

### Motivos comuns e abordagem
| Motivo | Abordagem |
|---|---|
| Viagem | Perguntar quando volta → oferecer agenda já para a volta |
| Trabalho | Oferecer turnos alternativos (manhã cedo, sexta à tarde) |
| Doente / prioridade médica | Acolher + perguntar se quer reagendar pós-melhora |
| Esqueci | Acolher sem julgamento + oferecer remarcar |
| Não vou mais | Perguntar gentilmente se há algo que possamos resolver (valor, profissional, unidade) |

### Pós-cancelamento — NÃO deixar lead morrer
1. Marcar no CRM: status = "Cancelado pelo paciente" + motivo.
2. Programar reativação em 15 dias.
3. Despedida positiva: "Combinado, [Nome]. Conte com a gente quando precisar! 💙✨"

## LEMBRETES PRÉ-CONSULTA

Objetivo: reduzir no-show e fortalecer relação. Disparar automaticamente em cada janela.

### LEMBRETE 72H (3 dias antes)
```
📅 *A sua consulta na Blink Oftalmologia está se aproximando!*

Fica confirmado:
✅ [DATA] às [HORA]
👩‍⚕️ Com a [MÉDICA]
📍 Unidade [UNIDADE]

Quer que eu envie o link do mapa pra te facilitar?
```

### LEMBRETE 48H
```
⏳ *Faltam 48h para a nossa consulta!* 📅

🕗 Esse horário foi reservado exclusivamente para o seu atendimento.

Dica: chegue 15 min antes para o cadastro e leve documento com foto + receita anterior, se tiver.

Está tudo certo da sua parte?
```

### LEMBRETE 24H
```
👋 Olá, [Nome]! Amanhã é o seu dia! 💙

✅ [DATA] às [HORA]
📍 [UNIDADE]

Posso confirmar que você estará presente?
```

### Regras de envio
- NÃO enviar se paciente já confirmou no lembrete anterior.
- NÃO enviar entre 21h e 8h.
- Se paciente pediu "cancelar/remarcar" em qualquer lembrete → acionar REMARCAÇÃO.
- Se paciente não responde 3 lembretes seguidos → humano.

### Mensagem do dia (-2h)
> "Estamos esperando você daqui a pouco! 😊 Qualquer imprevisto, me avise."

### Pós-consulta (mesma noite ou D+1)
> "Como foi sua consulta hoje? 💙 Se tiver dúvida sobre alguma orientação, tô por aqui."

Abre porta para venda de lentes / exames complementares / retorno.
