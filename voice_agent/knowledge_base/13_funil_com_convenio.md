# ATENDIMENTO COM CONVÊNIO — FLUXO OFICIAL

## 1. Identificação do tipo de consulta
Verificar SEMPRE o tipo de consulta do paciente para determinar cobertura.

### ❌ Procedimentos NÃO cobertos pelo convênio (mesmo se o plano for aceito)
- **SDP (Síndrome de Deficiência Postural)**
- **Casos de lentes de prisma**

### ✅ Procedimentos cobertos pelo convênio
- Consultas de rotina oftalmológica
- Avaliação para cirurgia
- Exames de diagnóstico

## 2. Verificação de cobertura
Consultar tabela de convênios (artigos 17 e 18) para confirmar tipos cobertos.

## 3. Processo de análise (passo a passo)
1. Verificar tipo de consulta
2. Consultar tabela de convênios
3. Informar paciente sobre cobertura e opções

**Conclusão:** Apenas SDP e lentes de prisma não são atendidas pelo convênio. Todos os demais tipos de consultas são cobertos, garantindo atendimento para a maioria das situações.

## 📅 LEVANTAMENTO DE DIA E HORÁRIO DE PREFERÊNCIA

Antes de transferir para o atendimento humano, solicitar informações para saber o dia e horário de preferência do paciente, utilizando os modelos abaixo:

### Unidade Asa Norte
```
Unidade de Atendimento: Asa Norte
Dias da semana: Segunda, Quarta e Sexta
Turno: Manhã e Tarde

### Unidade Águas Claras
```
Unidade de Atendimento: Águas Claras
